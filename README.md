# Expense Tracker (Python)

A simple command-line based expense tracker built using Python.
It allows users to record daily expenses, view spending history,
calculate total expenses, and filter expenses by category.

## Features
- Add expenses with amount, category, and note
- View all expenses
- Calculate total spending
- Filter expenses by category
- Data persistence using file handling

## Technologies Used
- Python
- File Handling
- Data Structures (List & Dictionary)

## How to Run
```bash
python expense_tracker.py
